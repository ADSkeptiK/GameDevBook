# Game Dev Challenges and Programming

Game Dev Challenges and Programming is a free, evolving programming book by ADSkeptiK.  
It is designed as a reference and learning resource for anyone interested in programming concepts, written in my own words, inspired by various sources which are cited throughout the book.

## Purpose
This book is currently in a pre-alpha stage. It is publicly available so that anyone can access it, learn from it, and provide feedback. The content may change frequently as it grows and develops.

## References
While the content is my own writing, many concepts were learned from these sources:
- Game Engine Architecture by Jason Gregory

Please see the inline citations and references section in the book for proper credit.
